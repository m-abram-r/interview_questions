
**1. Maximum Non-Decreasing Subsequence Length**

Given an array of integers representing monthly revenue, find the length of the longest subsequence (not necessarily contiguous) such that each element is greater than or equal to the previous one. A subsequence of length 1 is always valid.

*Example:* `[-1, 5, -8, 9, 7]` → `3`, since `[-1, 5, 9]` or `[-1, 5, 7]` are both valid non-decreasing subsequences of length 3.

---

**2. Sequence Join Point**

Define a sequence where each term is generated by adding the sum of its own digits to itself (e.g. 345 → 345 + 3 + 4 + 5 = 357). Given two starting integers, their sequences will eventually converge on a common value — call this the *join integer*. It is guaranteed that there exists a second, smaller integer (less than the join integer) whose sequence also reaches the join integer. Find the join integer for the two given starting values.

---

**3. Valid Triangle Sides**

Given three arrays `a`, `b`, and `c` of equal length, where each index represents the three side lengths of a potential triangle, return a boolean array of the same length indicating whether the values at each index satisfy the triangle inequality (i.e. could form a valid triangle).

---

**4. Minimum Router Radius**

Given two arrays `x` and `y` of equal length representing the coordinates of routers on a 2D plane, find the minimum radius `r` such that every router has at least one other router within distance `r` of it (i.e. no router is isolated).
